# Autonomous Object Detection & Path Following Robot

## Overview
This project simulates the basic working of a robot that can follow a path and avoid obstacles using simple decision-making logic.

Tech Stack
- Arduino (concept)
- IR Sensors (concept)
- Python (simulation)

Features
- Simulates line-following behavior
- Detects obstacles
- Decides movement direction

How It Works
- Sensor values are simulated
- Based on input, robot decides:
  - Move forward
  - Turn left/right
  - Stop if obstacle detected
